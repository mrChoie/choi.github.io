Choi's GameLib

The Choi's GameLib is a React web app project developed to initially test out React frameworks. The web app's concept is to have a catalogue of released video games sourced from the Video_Games.csv file.

This project is an on-going development for me to explore React frameworks. Information on this page may be updated from time to time.

Key Features

  • Single Page Application
  • CSV-powered library
  • Quick search
  • Paginated browsing
  • Catalogue Filtering

Planned Features

  • React Routing
  • User Management
  • User Uploads
  • Game Details

Tech Stack

  • Node.js (Runtime)
  • React Framework
  • TypeScript
  • HTML
  • CSS
  • Bootstrap
  • Git (version control)

Media

Homepage

A catalogue of games in paginated form with dynamic filters and a quick search bar

Single Page Application

The server only serves a single page html, and only the contents inside are dynamically changed without reloading the page.
