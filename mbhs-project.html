Malangas Budget Hotel System

The MBHS is designed to provide a light and comprehensive system for managing hotel operations, including reservations, room management, and hotel inventory management.

This application is intended for offline environment and doesnt require network functionalities

Key Features

  • User Authentication
  • Room Booking System
  • Room Management
  • Inventory Management
  • Reports & Transaction History
  • SQLite Database Integration

Tech Stack

  • C#
  • Windows Forms Application
  • SQLite

Media

Dashboard (main window)

Provides useful information such as expected check-ins and check-outs

User Management

Only authorized users can use the application

Sidebar

A collapsable sidebar UI to provide easy user navigation

All Reports (printable)

Reports of all recorded transactions

Singular Report (printable)

A Detailed report for a specific reservation

Check In Process

Month View Calendar

An interactive monthview calendar for determining active check-ins, pending reservation, and fully-booked days.
