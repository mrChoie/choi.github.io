SMC - Web Assist

The SMC Web Assist web application was designed with the primary goal of streamlining student consultations through a secure and user-friendly platform.

Inquiry/Concerns are recorded as tickets for easy tracking and reviewing

Key Features

  • Accessibility
  • Viewport Responsiveness
  • User Management
  • User Input Validation
  • Authentication & Authorization
  • Ticketing System
  • Ticket Conversation
  • Ticket Management
  • Password Reset Request
  • Email Notification

Tech Stack

  • Node.js (Runtime)
  • Express Framework
  • MySQL (Database)
  • JavaScript
  • HTML
  • CSS
  • Bootstrap

Authorization & Validation

Only Enrolled IDs are allowed. Duplicates are handled properly, and some inputs are strict with patterns such as email.

Ticket Submission

Users are opt to select categories based on the nature of their concern. After which they can monitor the status of their ticket in their profile dashboard.

Admin Roles

Admins have exclusive access to specific tickets based on their level/department. They can monitor tickets and take action through the ticket conversation
